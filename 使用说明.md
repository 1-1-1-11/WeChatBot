# 一键启动使用说明

## 🚀 三步开始使用

### 第 1 步：配置（首次使用）

双击运行：**配置向导.bat**

按提示输入：
- API 地址（如 `https://api.openai.com`）
- API 密钥（如 `sk-xxx`）
- 模型名称（推荐 `gpt-3.5-turbo`）
- 其他参数（直接回车使用默认值）

### 第 2 步：配置微信（首次使用）

1. 按 `Win + Ctrl + Enter` 启动 **Windows 讲述人**
2. 登录微信（如果未登录）
3. **等待 5 分钟**
4. 按 `Win + Ctrl + Enter` 关闭讲述人

> 💡 这是为了让微信 4.1+ 的 UI Automation 可见，只需配置一次。

### 第 3 步：启动

根据需要选择：

#### 🧪 测试模式（推荐新手）
双击：**快速启动.bat**
- 不会真实发送消息
- 只记录日志，安全测试

#### 🚀 生产模式（真实发送）
双击：**生产模式启动.bat**
- 会真实发送微信消息
- 确保已充分测试后再使用

## 🎮 控制台使用

启动后会显示控制台界面：

```
┌─────────────────────────────────────────────────┐
│ 微信值班状态：离线值班 | 模式：auto | 空闲：650秒 │
├─────────────────────────────────────────────────┤
│ [ ] 暂停自动回复                                │
│ (•) 自动检测  ( ) 强制在线  ( ) 强制离线        │
├─────────────────────────────────────────────────┤
│ [每日总览] [待办风险] [自动回复]                 │
└─────────────────────────────────────────────────┘
```

### 状态说明

- **在线汇总**：你正在使用电脑，只记录消息不自动回复
- **离线值班**：电脑空闲超过设定时间，允许自动回复

### 控制选项

- **暂停自动回复**：勾选后，即使离线也不会自动回复
- **自动检测**：根据鼠标键盘活动自动判断在线/离线（推荐）
- **强制在线**：始终视为在线，不自动回复（调试用）
- **强制离线**：始终视为离线，允许自动回复（测试用）

### 标签页

- **每日总览**：每晚 22:00 自动生成的消息汇总
- **待办风险**：中风险消息（包含"报价"、"转账"等关键词）
- **自动回复**：已发送的自动回复记录

---

## ✅ 自动回复触发条件

**同时满足以下所有条件才会自动回复**：

- ✅ 电脑空闲时间 ≥ 设定的空闲分钟数（默认 10 分钟）
- ✅ 收到的是个人消息（不是群聊）
- ✅ 消息是低风险（不包含"报价"、"转账"、"微信支付"等关键词）
- ✅ 距离上次回复该联系人已超过冷却时间（默认 10 分钟）
- ✅ 未勾选"暂停自动回复"
- ✅ DRY_RUN=false（生产模式）

## 🧪 测试流程建议

### 测试模式验证（安全）

1. 确保 `DRY_RUN=true`
2. 运行 `快速启动.bat`
3. 让电脑空闲 10 分钟
4. 让测试联系人发消息："在吗"
5. 20-60 秒后，查看控制台"自动回复"标签页
6. 应该看到"准备发送"的记录（但实际不会发送）

### 生产模式验证（谨慎）

1. 确认测试模式验证成功
2. 运行 `生产模式启动.bat`
3. 重复上述步骤
4. 测试联系人会收到自动回复消息

## ❓ 常见问题

### Q1: 双击 bat 文件没反应或闪退？

**解决方法**：
- 右键 → "以管理员身份运行"
- 或在 PowerShell 中运行：`.\快速启动.bat`

### Q2: 提示"未找到 Python"？

**解决方法**：
1. 下载安装 Python 3.9+：https://www.python.org/downloads/
2. 安装时勾选"Add Python to PATH"
3. 重启命令行窗口

### Q3: 为什么不自动回复？

**检查清单**：
- [ ] 电脑空闲时间够了吗？（默认 10 分钟）
- [ ] 是个人消息吗？（群聊不会回复）
- [ ] 消息内容是低风险吗？（不包含"报价"等关键词）
- [ ] `DRY_RUN=false` 吗？（测试模式不会真实发送）
- [ ] 勾选了"暂停自动回复"吗？

### Q4: 提示"微信 UI Automation 不可用"？

**解决方法**：按照第 2 步重新配置微信（讲述人模式）

### Q5: 如何停止运行？

- 关闭控制台窗口（点击 X）
- 或在控制台窗口按 `Ctrl + C`

## ⚙️ 修改配置

如需修改配置：

1. 直接编辑 `.env` 文件（用记事本打开）
2. 或重新运行 `配置向导.bat`

**主要参数**：

```env
IDLE_MINUTES=10                      # 空闲多少分钟触发离线
AUTO_REPLY_DELAY_MIN_SECONDS=20      # 最小延迟
AUTO_REPLY_DELAY_MAX_SECONDS=60      # 最大延迟
CONTACT_COOLDOWN_MINUTES=10          # 同一联系人冷却时间
DRY_RUN=true                         # true=测试，false=生产
```
