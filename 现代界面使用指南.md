# 🎨 现代化界面使用指南

## ✨ 新界面特性

基于你提供的 Gemini 智能体界面设计，我创建了全新的**深色主题现代化控制台**：

### 🎯 核心特性

1. **深色主题**：护眼的深色配色方案
2. **左右分栏布局**：
   - **左侧边栏**：状态卡片、控制面板、统计信息
   - **右侧主区**：消息记录、标签页
3. **实时状态指示**：彩色圆点显示在线/离线/错误状态
4. **快速模式切换**：类似 Gemini/自动 的快捷切换
5. **卡片式设计**：现代化的 UI 组件

## 🛠️ 自定义配色

如需修改配色，编辑 `wechat_bot/modern_dashboard.py` 的顶部常量：

```python
class ModernDashboard:
    # 深色主题配色
    BG_DARK = "#1e1e1e"        # 主背景
    BG_SIDEBAR = "#252526"     # 侧边栏背景
    BG_CARD = "#2d2d30"        # 卡片背景
    ACCENT_GREEN = "#4ec9b0"   # 在线状态
    ACCENT_ORANGE = "#ce9178"  # 离线状态
    ACCENT_RED = "#f48771"     # 错误状态
```

## 📝 使用技巧

### 快速切换模式
点击左侧"运行模式"区域的单选按钮即可快速切换：
- 开发调试 → 选择"强制在线"（不会自动回复）
- 正式值班 → 选择"自动检测"（根据空闲时间）
- 测试回复 → 选择"强制离线"（始终触发回复）

### 监控实时状态
观察左上角的**状态指示灯**：
- 🟢 绿色闪烁 = 正在处理消息
- 🟠 橙色常亮 = 离线值班中
- 🔴 红色 = 出现错误，查看日志

### 查看详细日志
切换右侧标签页：
- **💬 自动回复** → 查看最近 20 条回复记录
- **⚠️ 待办风险** → 查看需要人工处理的消息
- **📊 每日总览** → AI 生成的每日工作总结

## 🐛 故障排查

### 问题 1：界面显示异常
**解决**：确保安装了 tkinter
```powershell
# Windows 一般自带，如果缺失重新安装 Python 并勾选 tcl/tk
```

### 问题 2：统计数据不更新
**解决**：界面每 2 秒自动刷新，检查是否有错误提示

### 问题 3：想恢复旧界面
**解决**：启动时添加 `--classic-ui` 参数
```powershell
python -m wechat_bot --env .env --classic-ui
```

## 📚 技术细节

### 线程安全设计
- 使用 `threading.Lock` 保护共享状态
- UI 刷新在主线程，消息处理在工作线程
- 避免 Tkinter 跨线程访问问题

### 性能优化
- 日志最多显示 20 条（避免内存占用）
- 2 秒刷新间隔（平衡实时性和性能）
- 文本控件禁用编辑（减少重绘）
